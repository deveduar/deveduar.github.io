## JavaScript Object vs JSON

### Diferencias Clave
- **JSON**: Siempre requiere comillas dobles en las keys
- **JS Object**: No siempre necesita comillas dobles, puede terminar con coma
- **JS Object**: Permite acceso individual a los datos
