# Session Initiation Protocol (SIP)

## Funcionamiento  
- [autenticacion](/autenticacion/autenticacion/)
- **Mensajes**: INVITE, BYE, ACK.  
- **Registrar**: Autenticación de usuarios.  

## Ejemplo de Llamada  
1. Cliente A → INVITE → Servidor SIP.  
2. Servidor → 200 OK → Cliente B.
