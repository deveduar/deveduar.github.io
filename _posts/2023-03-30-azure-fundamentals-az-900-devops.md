## Programa de 12 Meses de Azure Cloud (Cuenta Gratuita)

- Azure ofrece una **cuenta gratuita** que permite explorar y aprender la plataforma sin coste inicial.
- Durante los **primeros 30 días**, se dispone de un **crédito de 200 USD** para probar cualquier servicio de Azure sin restricciones.
- Al [pasar a precios de pago por uso](https://learn.microsoft.com/en-us/azure/cost-management-billing/manage/upgrade-azure-subscription?wt.mc_id=rmskilling_azclr_onboarding_inproduct_gdc) dentro de los 30 días o tras consumir el crédito:
	- Se desbloquean **servicios gratuitos durante 12 meses**.
	- Se mantiene el acceso a servicios con cuotas gratuitas mensuales.
- Se incluyen **más de 55 servicios gratuitos** incluso después de los 12 meses iniciales:
	- Lista completa: [Servicios gratuitos de Azure](https://azure.microsoft.com/en-us/free/#all-free-services?wt.mc_id=rmskilling_azclr_onboarding_inproduct_gdc)
- Los servicios gratuitos:
	- Pueden utilizarse en **entornos de producción**, no solo para desarrollo o pruebas.
	- Permiten crear soluciones reales como:
		- Aplicaciones web escalables.
		- Aplicaciones móviles.
		- Soluciones con IA.
		- Modernización de aplicaciones con contenedores.
- Si no se cambia a pago por uso tras los 30 días o al agotar los 200 USD:
	- No se genera ningún cargo.
	- La **suscripción y los recursos se deshabilitan automáticamente**.

### Azure Pass (Crédito Promocional)

- Azure Pass proporciona **100 USD de crédito** para prácticas y laboratorios.
- Proceso general:
	1. Solicitar el **Azure Pass** mediante el enlace correspondiente.
	2. Las unidades son limitadas y deben reclamarse antes del **31/12/2022**.
	3. El código debe activarse antes del **14/02/2023**.
	4. El crédito:
		- Tiene una duración máxima de **30 días** desde la activación.
		- Se desactiva al agotarse el saldo.
- Ideal para:
	- Laboratorios prácticos.
	- Preparación de certificaciones.
	- Pruebas de servicios sin coste personal.
