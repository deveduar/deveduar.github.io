## Código — Ejemplo Jenkinsfile Declarativo
{% raw %}
```groovy
pipeline {
agent any

stages {
	stage('Build') {
		steps {
			sh 'npm install'
		}
	}
	stage('Test') {
		steps {
			sh 'npm test'
		}
	}
	stage('Docker Build') {
		steps {
			sh 'docker build -t app:latest .'
		}
	}
	stage('Deploy') {
		steps {
			sh './deploy.sh'
		}
	}
}

post {
	success {
		echo "Pipeline finalizado con éxito."
	}
	failure {
		echo "Pipeline fallido."
	}
}
}
```
{% endraw %}`
