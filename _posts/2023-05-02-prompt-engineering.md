## Conceptos Clave

- **Prompt Engineering**: Técnica de diseño de instrucciones para que modelos de lenguaje generen resultados más precisos, relevantes o creativos.
- **Objetivo**: Maximizar la utilidad del modelo mediante prompts claros, estructurados y contextuales.
- **Elementos de un buen prompt**:
	Tipo de respuesta deseada (resumen, lista, explicación detallada, código, etc.)
	Contexto relevante (información adicional para guiar la respuesta)
	Estilo y tono (formal, informal, técnico, creativo)
	Restricciones (longitud, formato, exclusiones)

## Buenas Prácticas

- Ser específico y directo en la solicitud.
- Proporcionar contexto suficiente para evitar ambigüedad.
- Usar ejemplos dentro del prompt para guiar el resultado.
- Iterar y refinar prompts para mejorar resultados.
- Experimentar con la estructura: preguntas abiertas vs instrucciones paso a paso.
- Aplicar técnicas de **chain-of-thought** cuando se requiera razonamiento complejo.

## Técnicas Avanzadas

- **Few-shot prompting**: Incluir ejemplos de entrada-salida en el prompt para que el modelo imite el patrón.
- **Zero-shot prompting**: Solicitar una tarea sin ejemplos, confiando únicamente en la descripción.
- **Instruction tuning**: Ajustar prompts para que el modelo siga instrucciones específicas con precisión.
- **Role prompting**: Asignar un rol al modelo (ej. “Eres un experto en biología”) para contextualizar respuestas.
- **Prompt chaining**: Dividir tareas complejas en varios prompts encadenados para mejorar la coherencia y exactitud.
- **Expanding prompts**: Añadir detalles y contexto para enriquecer la respuesta.
- **Transforming prompts**: Cambiar la perspectiva, formato o estilo de la respuesta solicitada.

## Aplicaciones Prácticas

- **Resúmenes automáticos**: Transformar textos extensos en resúmenes claros y concisos. l4-summarizing
- **Inferencias y deducciones**: Generar conclusiones a partir de información proporcionada. l5-inferring
- **Chatbots especializados**: Crear agentes conversacionales con roles específicos y conocimiento focalizado. l8-chatbot
- **Transformación de datos o texto**: Reformatear, reescribir o estructurar información según necesidades. l6-transforming
- **Expansión de contenido**: Ampliar textos o ideas manteniendo coherencia y estilo. l7-expanding
- **Optimización de prompts**: Ajustar iterativamente prompts para mejorar precisión, creatividad o utilidad.
