# Frontend — Temas Pendientes 2025

- Medición de performance con Web Vitals y Lighthouse.

- Persistencia de estado entre sesiones y tabs.  
- Caching avanzado de datos para aplicaciones SPA.  
- Librerías: Redux Persist, React Query, Apollo Cache, Pinia Plugins.

## Ejemplo: GraphQL Subscription (React + Apollo)

{% raw %}
```javascript
import { useSubscription, gql } from '@apollo/client';

const MESSAGE_SUBSCRIPTION = gql`
	subscription OnMessageAdded {
		messageAdded {
			id
			content
			author
		}
	}
`;

function Chat() {
	const { data, loading } = useSubscription(MESSAGE_SUBSCRIPTION);

	if (loading) return <p>Cargando mensajes...</p>;
	return (
		<ul>
			{data.messageAdded.map(msg => (
				<li key={msg.id}>{msg.author}: {msg.content}</li>
			))}
		</ul>
	);
}
```
{% endraw %}
