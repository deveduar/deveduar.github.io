# Omnivore Powershell

```base
type: list
name: "Notas con #powershell en Omnivore"
order:
  - property: date_saved
    direction: desc
columns:
  - file.name
  - date_saved
filters:
  and:
    - file.inFolder("Omnivore")
    - file.hasTag("powershell", "Powershell")
views:
  - type: table
    name: Table
    sort:
      - property: file.mtime
        direction: DESC
```
