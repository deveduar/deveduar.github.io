## Conceptos Fundamentales de IA y ML

### Regresi√≥n
- La **regresi√≥n** modela la relaci√≥n entre una o m√°s variables de entrada (*features*) y una variable de salida (*target*).
	- Se usa para **predecir valores num√©ricos**: precios, temperaturas, probabilidades.
	- T√©cnicas comunes: regresi√≥n lineal, polin√≥mica, regularizaci√≥n (Ridge, Lasso).
	- Objetivo: encontrar la funci√≥n que mejor minimice el error entre predicci√≥n y realidad.

### √Årboles de decisi√≥n
- Un **√°rbol de decisi√≥n** divide el espacio de decisiones mediante condiciones.
	- Ideal para clasificaci√≥n o regresi√≥n.
	- Genera reglas tipo: *‚Äúsi edad < 18 y compra = no ‚Üí categor√≠a X‚Äù*.
	- Ventajas: interpretables, pueden manejar datos categ√≥ricos o num√©ricos.
	- Limitaciones: tienden a sobreajustar si no se podan.

### M√©todos Bayesianos
- Basados en la **probabilidad de Bayes**, permiten actualizar creencias seg√∫n evidencia.
	- Usados para clasificaci√≥n de textos (spam vs no spam), detecci√≥n de patrones, filtrado.
	- El modelo calcula la **probabilidad posterior** de una clase dadas sus caracter√≠sticas.
	- Ejemplo: *Naive Bayes*, que asume independencia entre caracter√≠sticas.

### Agrupaci√≥n (Clustering)
- T√©cnicas para **agrupar elementos sin categor√≠as previas**.
	- Busca similitud entre elementos (distancias, densidades, conectividad‚Ä¶)
	- Ejemplos: K-means, DBSCAN, clustering jer√°rquico.
	- Usos: segmentaci√≥n de clientes, agrupaci√≥n de im√°genes, estructuraci√≥n de datos.

## An√°lisis del Problema (Workflow)

Proceso t√≠pico antes de aplicar ML o IA a un dataset.

### An√°lisis de datos
- Exploraci√≥n inicial: tendencias, outliers, valores nulos.
- Validaci√≥n de la consistencia.
- Identificaci√≥n de ruido o errores.
- Evaluaci√≥n de correlaciones y distribuci√≥n.

### Preparaci√≥n de datos
- Selecci√≥n de variables relevantes.
- Construcci√≥n de variables adicionales (*feature engineering*).
- Normalizaci√≥n, limpieza, codificaci√≥n categ√≥rica.
- Scripts que automatizan el preprocesamiento.
- Formateo del dataset para su uso en modelos.

### Modelado
- Selecci√≥n de algoritmos seg√∫n el tipo de problema:
	- regresi√≥n, clasificaci√≥n, clustering, redes neuronales.
- Consideraci√≥n de requerimientos, complejidad y contexto.
- Entrenamiento, ajuste de hiperpar√°metros, validaci√≥n cruzada.

### Evaluaci√≥n
- Comparaci√≥n entre predicciones y valores reales.
- Detecci√≥n de problemas comunes:
	- falta de datos,
	- variables mal seleccionadas,
	- modelo inadecuado.
- M√©tricas seg√∫n el modelo: precisi√≥n, recall, MSE, F1, ROC-AUC.

### Implementaci√≥n y comunicaci√≥n
- Documentaci√≥n del proceso y visualizaciones.
- Presentaci√≥n clara para clientes o stakeholders.
- Conclusiones, reportes, recomendaciones y pr√≥ximos pasos.
